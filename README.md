# 🎲 LuckyRoll

**LuckyRoll** — tam stack zər lotereyası oyunu. İstifadəçilər qeydiyyatdan keçir, coin mərc edir, müxtəlif oyun rejimləri ilə oynayır və liderboardda rəqabət aparır.

---

## 🛠 Tech Stack

| Hissə | Texnologiya |
|---|---|
| Frontend | React 18, TypeScript, Vite |
| Styling | CSS Modules |
| Backend | Django 5, Django REST Framework |
| Auth | Token-based (DRF) |
| Database | SQLite |

---

## 🎮 Oyun Rejimləri

| Rejim | Qaydalar |
|---|---|
| 🎲 Klassik Zər | Yuxarı 25% → 3x \| Yuxarı 50% → 1.5x |
| 🍀 Şanslı 7 | Cəm = 7 → 5x \| 7 var → 2x |
| 💎 Cekpot | Hamısı eyni → 10x \| Yuxarı 20% → 3x |

---

## ✨ Xüsusiyyətlər

- 🔐 Token əsaslı qeydiyyat və giriş sistemi
- 🎲 D4 / D6 / D8 / D10 / D12 / D20 zər növləri
- 🔢 1–6 zər sayı seçimi
- 🪙 Coin sistemi — mərc et, qazan, xərcle
- 🎁 Gündəlik bonus sistemi (24 saatlıq geri sayım)
- 🏆 Liderboard — oyunçuları ümumi uduşa görə sırala
- 📊 Statistika — atış sayı, ən yaxşı cəm, ortalama, üz tezliyi qrafiki
- 📜 Oyun tarixi
- 🌙 Qaranlıq / İşıqlı tema
- 🎉 Böyük uduşlarda konfeti animasiyası
- 📱 Responsiv dizayn (mobil dəstək)
